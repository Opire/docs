# 常见问题解答

<br/>
<InfoPanel>
    如果您在使用 Opire 时有任何问题或遇到任何问题，您可以通过我们的 Discord 的 `#ask-us` 频道与我们联系，我们将尽快回答。
</InfoPanel>
<br/>

<details>
    <summary><strong>Opire 可以用于私有项目吗？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        是的，我们的命令可以在私有项目中使用，但有一个细节需要考虑。
        
        创建的奖励将显示在 Opire 平台上，但是没有访问权限的开发人员将无法尝试解决问题或领取奖励。
        
        如果外部人员想要合作，他们需要直接与项目管理员沟通以请求访问权限。
    </div>
</details>

<details>
    <summary><strong>如果我没有获得奖励怎么办？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        如果您发现自己处于这种情况下，我们为发生的事情感到抱歉 💔
        
        首先，请尝试与奖励创建者联系。他们可能没有意识到他们已经可以支付您，他们可能已经忘记了，或者他们还没有能够这样做。我们相信社区及其通过对话解决冲突的能力。

        Opire 是为社区设计的工具，我们希望它被正确使用。我们理解情况并非总是如此，在用户创建了许多奖励但不打算支付的情况下，Opire 有算法检测此类行为并禁用用户以防止对社区造成进一步伤害。这些用户将无法创建更多奖励，直到处理未支付的奖励的付款。
    </div>
</details>

<details>
    <summary><strong>Opire 适用于谁？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        几个群体可以从 Opire 中受益：

        - <strong>开发者：</strong> 这是一个赚取额外收入的机会，而无需受时间表或繁琐的程序的束缚。此外，您将通过解决不同语言中的真实问题来增强您的技能。
        - <strong>公司：</strong> 我们在不涉及您的开发团队的情况下，简化了您项目依赖项的问题解决过程。因此，您可以通过将资源集中在为公司增加价值的方面来优化成本。
        - <strong>开源代码所有者：</strong> 您可以利用 Opire 增加您的社区，并通过其他开发人员的贡献保持项目的活力。此外，如果其他人为您的项目创建奖励，您将获得额外的收入，以感谢您对开源世界的影响。
        - <strong>开源爱好者：</strong> 与捐赠平台不同，Opire 上的奖励与实际项目问题相关，直接影响代码质量的提高。您不会成为匿名捐赠者，而是将积极参与项目的改进和增长。
    </div>
</details>

<details>
    <summary><strong>如果我收到收入，我如何申报收益？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        由于您可以从世界各地获得收入，我们无法提供适合每个国家要求的解决方案（这超出了我们的范围）。
        
        收到收入的人负责遵守其国家的法律，根据需要申报收入。您可以直接从您的 Stripe 帐户获取收入列表。
    </div>
</details>

<details>
    <summary><strong>如果我 想创建奖励，我是否需要配置 Stripe？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        不需要。
        
        您只需要在支付您之前创建的奖励时输入支付方式。
    </div>
</details>

<details>
    <summary><strong>作为开发人员，如果我在领取奖励时没有配置我的 Stripe 帐户会发生什么？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        如果有人尝试支付您，但您的 Stripe 帐户未配置，支付无法进行，我们将通知您配置您的帐户。配置后，请联系奖励创建者要求重新尝试支付过程。
    </div>
</details>

<details>
    <summary><strong>我可以在哪些国家使用 Opire？</strong></summary>
    <div style={{ marginTop: '10px' }}>
        Opire 并未明确禁止在任何国家使用。但是，当使用 Stripe 等支付提供程序时，用户受到 Stripe 强加的条件约束。

        在创建奖励之前，请确保您的国家在 Stripe 上受支持。
    </div>
</details>

<details>
    <summary><strong>如果 Opire 消失了，我的钱会怎么样？</strong></summary>
    <div style={{ marginTop: '10px' }}>
       你不会失去一分钱。

       在 Opire，我们不会保留未来使用的资金。您的钱始终掌握在您手中，直到您决定支付开发者为止。
       
       我们也不会要求您的信用卡或银行信息。我们无权访问这些数据（也不希望访问），因此不会有任何意外的付款。
       
       当您在 Opire 上进行支付时，所有事务都通过 Stripe 处理，开发者直接收到他们的份额。 Opire 在处理奖励/小费支付时只保留相应的％作为我们的费用。
    </div>
</details>
