# Instalar app

Para permitir a los usuarios interactuar con Opire directamente dentro de los repositorios de su plataforma favorita (GitHub), es necesario que el repositorio tenga instalada una aplicación que habilite el uso de ciertos [comandos](https://docs.opire.dev/es/overview/commands) dentro de las issues y PRs.

Esta aplicación gestiona la comunicación con la API de Opire, de forma que lo que se lleve a cabo a través de los comandos se verá repercutido en [Opire](https://app.opire.dev/).
