# Instalar Bot

Para permitir que os usuários interajam com Opire diretamente dentro de seus repositórios de plataforma favoritos (GitHub), é necessário que o repositório tenha um bot instalado que permita o uso de certos [comandos](/overview/commands) dentro de problemas e PRs.

Este bot gerencia a comunicação com a API do Opire, garantindo que as ações realizadas por meio de comandos sejam refletidas no [Opire](https://app.opire.dev/).
