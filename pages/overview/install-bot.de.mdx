# Bot installieren

Um Benutzern zu ermöglichen, direkt innerhalb ihrer bevorzugten Plattform-Repositories (GitHub) mit Opire zu interagieren, ist es erforderlich, dass das Repository eine installierte Anwendung hat, die die Verwendung bestimmter [Befehle](/overview/commands) innerhalb von Problemen und Pull Requests ermöglicht.

Diese Anwendung verwaltet die Kommunikation mit der Opire-API und stellt sicher, dass Aktionen, die durch Befehle ausgeführt werden, in [Opire](https://app.opire.dev/) reflektiert werden.
