# 安装机器人

为了让用户可以直接在他们喜爱的平台存储库（GitHub）内与 Opire 进行交互，需要在存储库中安装机器人，以启用在问题和 PR 中使用某些[命令](/overview/commands)。

该机器人管理与 Opire API 的通信，确保通过命令执行的操作将在[Opire](https://app.opire.dev/)中反映出来。
