# Install App

To allow users to interact with Opire directly within their favorite platform repositories (GitHub), it's necessary for the repository to have an installed application that enables the use of certain [commands](https://docs.opire.dev/overview/commands) within issues and PRs.

This application manages communication with the Opire API, ensuring that actions performed through commands will be reflected in [Opire](https://app.opire.dev/).
