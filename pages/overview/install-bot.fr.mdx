# Installer Bot

Pour permettre aux utilisateurs d'interagir avec Opire directement dans leurs dépôts de plateforme préférés (GitHub), il est nécessaire que le dépôt ait une bot installée qui permet l'utilisation de certains [commandes](/overview/commands) dans les problèmes et les PR.

Cette bot gère la communication avec l'API Opire, garantissant que les actions effectuées via les commandes soient reflétées sur [Opire](https://app.opire.dev/).
