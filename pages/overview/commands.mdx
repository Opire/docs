# Commands

`/reward 20` create a 20 euros reward for the issue where the comment is writted, anyone can create rewards if the organization or repository has the MakeMyChange app installed. Anyone can create a lot of rewards for the same issue

`/try` indicates that you are going to try to do the issue to earn the reward. If another developer is trying to earn the reward we notify you with the others developers trying in case you want to do it together.

`/claim #17` claim the reward for the issue number 17

`/tip 20 @nabby27` this create a 20 euros tip for the user @nabby27. You can pay it going to MakeMyChange
