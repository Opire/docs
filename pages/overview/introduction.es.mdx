# Introducción

## ¿Qué es Opire?

Opire es una plataforma de **recompensas para desarrolladores de software**.

Este proyecto nace con <u>un objetivo claro</u>: que **cualquiera pueda vivir del software de código abierto**. Tanto programadores que quieran centrarse en desarrollar software y mejorar código libre, como propietarios y mantenedores de código abierto que aporten a la comunidad con sus proyectos. 

Con Opire **cualquiera puede crear recompensas en proyectos de código abierto** y hacer crecer su comunidad, mientras que los desarrolladores pueden resolver las issues y **ganar las recompensas asociadas**.

## ¿Cómo funciona?

Opire se compone de dos piezas: una [app](/es/overview/install-app) (a instalar en GitHub), y una [plataforma web](https://app.opire.dev).

La app permite la utilización de diferentes [comandos](/es/overview/commands) integrados directamente con GitHub para interactuar con la plataforma: crear recompensas, reclamarlas, etc.

La plataforma web se centra en la visualización y gestión de las recompensas. Entre otras funcionalidades permite la búsqueda avanzada (por tecnología, precio...), el pago de las que ya hayan sido reclamadas, el histórico de las que has resuelto, configuración de tu perfil, etc.

Para ver en detalle el ciclo de vida de una recompensa, revisa esta [sección](/es/rewards/workflow).
